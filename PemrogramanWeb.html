Tertarik belajar Web Programming, Silahkan daftar.



HTML



HTML adalah tag tag yang digunakan untuk membuat kerangka dari sebuah Web seperti text, audio, video, picture dan pengorganisasian Web.

CSS



CSS adalah script-script yang digunakan untuk memperindah tampilan dari web seperti pengaturan warna, teks dan layout sebuah web.

JavaScript



JavaScript adalah bahasa pemrograman yang dapat membuat Website menjadi interaktif, pengaturan input-output serta animasi.

PHP

PHP adalah bahasa pemrograman untuk mengatur interaksi Web dengan database.

Laravel

Laravel adalah framework MVC untuk PHP. Memudahkan developer dalam memisahkan front-end dan back-end.
